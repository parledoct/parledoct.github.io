Parledoct (like 'aqueduct') is a collection of tutorials and supporting tools to assist in the deployment of speech processing pipelines for language documentation projects.

These tutorials are aimed at those who have some basic familiarity with speech data (e.g. annotating in Praat/ELAN) and scripting (e.g. in Praat or R) and want to learn to directly use modern speech processing tools via the command-line interface, complementing projects such as Elpis, which provides a user-friendly interface for similar tools.

The tutorials fall into one of three categories:

Processing pre-prepared speech data on Google Colab. These tutorials provide self-paced introductions to performing various speech processing tasks (e.g. voice activity detection) on Google Colab with pre-prepared datasets.

Dataset pre- and post-processing. These tutorials provide introductions to preparing data from language documentation projects for use with various tools (e.g. stereo-to-mono conversion, re-sampling) and exporting the output of various tools into desired formats (e.g. ELAN .eaf files).

Speech processing in the cloud. As the compute power of your laptop or desktop is sometimes not enough to perform some tasks, these tutorials provide introductions to using cloud providers such as Google Cloud for performing these tasks.
