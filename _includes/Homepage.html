Parledoct (like 'aqueduct') is a collection of tutorials and supporting tools to assist in the deployment of speech processing pipelines for language documentation projects. These tutorials are aimed at those who have some basic familiarity with speech data (e.g. annotating in Praat/ELAN) and scripting (e.g. in Praat or R) and want to learn to directly use modern speech processing tools via the command-line interface, complementing projects such as Elpis which provides a user-friendly interface for similar tools.

The tutorials fall into one of two main categories: concepts and recipes. The recipes will be kept relatively brief and contain links to relevant conceptual introductions, which are more expository (note: currently, many of them are placeholders/external links). Earlier tutorials provide scaffolding for the later ones: for example, you first learn to perform a relatively simple operation over a set of files (loop through audio files in a folder and calculate the duration for each of them) after which a more complex operation is introduced (voice activity detection and exporting detected regions to an ELAN .eaf file).
